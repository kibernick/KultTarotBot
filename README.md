# KultTarotBot
Very simple Python-based Discord bot for drawing Tarot cards in the Kult Divinity Lost roleplaying game.

Usage (in Discord): 
- `!tarot` - draws 5 tarot cards.
- `!tarot n` - draws n tarot cards, where n is from 1 to 10 inclusive.
